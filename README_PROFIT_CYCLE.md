# 🚀 БАЛТСЕТЬ - Полный цикл прибыли

## 💰 Модель монетизации

**Комиссия платформы: 25%**  
**Выплата мастеру: 75%**

---

## 📊 Цикл получения прибыли

### 1️⃣ Клиент заказывает услугу
📍 **Страница:** `/catalog.html`

- Клиент выбирает услугу из каталога
- Нажимает кнопку "Заказать"
- Услуга добавляется в корзину
- **Автоматический переход** → `/payment.html`

---

### 2️⃣ Оплата заказа
📍 **Страница:** `/payment.html`

**Что происходит:**
- Отображается корзина с выбранными услугами
- Автоматический расчёт:
  - Сумма услуг
  - Комиссия платформы (25%)
  - Выплата мастеру (75%)
- Клиент видит разбивку средств
- Выбор способа оплаты:
  - 💳 Банковская карта
  - 📱 СБП
  - 💵 Наличные мастеру

**Пример расчёта:**
```
Установка розеток: 2 шт × 500₽ = 1 000₽
Замена проводки: 1 шт × 2 000₽ = 2 000₽
─────────────────────────────────────
ИТОГО: 3 000₽

Распределение:
├─ Мастер получит (75%): 2 250₽
└─ Комиссия платформы (25%): 750₽
```

**После оплаты:**
- Заказ сохраняется в `localStorage`
- Отправка уведомления в Telegram бот
- Назначение мастера

---

### 3️⃣ Мастер выполняет работу
📍 **Страница:** `/order-complete.html`

**Что происходит:**
- Мастер получает заказ через Telegram бота `@ai_service_master_bot`
- Выезжает к клиенту
- Выполняет работы
- **Подтверждение выполнения:**
  - Загрузка минимум 2 фото выполненных работ
  - Комментарий к работе (опционально)
  - Отправка подтверждения клиенту

**Информация для мастера:**
```
Заказ #123456
Клиент: Иван Петров
Услуга: Установка розеток
Сумма заказа: 2 000₽

💰 Ваша выплата: 1 500₽ (75%)
```

---

### 4️⃣ Клиент подтверждает работу
📍 **Telegram бот:** `@ai_service_client_bot`

- Клиент получает уведомление с фото
- Проверяет качество работ
- Подтверждает выполнение
- **Деньги автоматически разделяются:**
  - 75% → Мастеру на счёт
  - 25% → Платформе (прибыль)

---

### 5️⃣ Статистика и прибыль
📍 **Страница:** `/admin.html`

**Админ-панель показывает:**

#### 📊 Основные метрики:
- 💰 **Общий оборот** — вся сумма заказов
- 🏦 **Комиссия платформы (25%)** — чистая прибыль
- 👷 **Выплачено мастерам (75%)** — расходы
- 📦 **Всего заказов** — количество

#### 📈 Динамика оборота:
- График за последние 7 дней
- Визуализация роста прибыли

#### 📋 История заказов:
Таблица всех заказов с детализацией:
- Номер заказа
- Дата и время
- Список услуг
- Сумма заказа
- Комиссия (25%)
- Выплата мастеру (75%)
- Статус (Оплачено/В обработке/Отменено)

#### 📥 Экспорт данных:
- Выгрузка в Excel/CSV
- Фильтры по периоду и статусу
